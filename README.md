# Automação de API - Serverest

Projeto de automação de testes de API utilizando Postman e Newman para validação de endpoints REST da API Serverest.

---

## Tecnologias utilizadas

- Postman
- Newman
- JavaScript
- Node.js
- Git
- GitHub

---

## Funcionalidades automatizadas

### Login
- Realizar login com sucesso
- Validar retorno do token

### Produtos
- Criar produto
- Buscar produto
- Atualizar produto
- Deletar produto

### Usuários
- Criar usuário

---

## Validações realizadas

- Status code
- Mensagens de sucesso
- Validação de ID
- Validação de campos da resposta
- Fluxo CRUD completo

---

## Como executar o projeto

### Pré-requisitos

- Node.js instalado
- Newman instalado globalmente

### Instalar Newman

```bash
npm install -g newman
